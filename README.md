# Voxel Sandbox Engine

A modern, data-driven voxel engine written in Java.

## Vision
This project aims to build a scalable voxel engine
with a database-first architecture and spatial querying.

## Key Principles
- Engine / Content separation
- Database-driven world
- Streaming & spatial indexing
- Performance-first design
